# Night Sky App Backend

Night Sky is MERN stack application. It's playground which allows user to manipulate how the sky looks at night.

## Installation

Download repository from github.
Install node modules using npm.
```bash
npm install
```
Set up env file so app can connect to database.

## Usage
You should be using GUI made for this application in react.
If not you can manipulate how the "sky" looks in db using endpoints (add, edit, delete constellations and stars).
